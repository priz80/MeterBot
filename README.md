Телеграм бот для записи показаний приборов учета потребления ресурсов(газ, вода, электричество).
Функционал:
- внесение данных
- история всех внесенных данных
- автоматический рассчет объема потребления
- автоматический рассчет среднего объема потребления
- удаление данных
Текущая версия для личного пользования.
Запуск - создать файл .env с содержанием BOT_TOKEN = 'your key'
